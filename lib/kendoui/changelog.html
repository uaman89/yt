Release notes for Kendo UI Professional v2016.1.226

Components

What's fixed

  • Compiled AMD modules no longer export `kendo`
  • minified scripts require "jquery.min" - they should require "jquery"
  • Wrong k-content background-color in Moonlight theme
  • Popup in IE11 is not positioned correctly when placed in a scrolled container (jQuery 1.12.0/2.2.0)
  • AMD module names are incorrect in the minified files available on CDN
  • Font icons are distorted when exporting to PDF
  • Checkboxes move scrollbar on parent container in IE / Chrome

AngularJS

What's fixed

  • Compile itemTemplate to the dataSource item
  • Widgets are not destroyed correctly during navigation via angular-ui-router
  • Widget with remote binding selects the first item and updates the model incorrectly on first load

What's new

  • Support for different string formats for disableDates array

What's fixed

  • Missing styles for k-state-disabled in Nova theme
  • Content shrinks by 10px at the end of decade-year-month animations
  • Review interactive states in Calendar

Chart

What's fixed

  • JavaScript error when calling refresh immediately after initialization (regression from internal build)
  • First bar from series lacks fill when the color function returns undefined
  • Radar series do not arrange points according to categoryAxis binding
  • "Maximum stack exceeded" error might appear when using Charts in Angular (regression from internal builds)
  • Error when loading kendo.dataviz.chart with Require JS ("Color is not a function")
  • Slowdown in charts with multiple empty categories
  • Pan and zoom does not work for scatter charts with multiple axes
  • Chart layout may be incorrect if using a custom font
